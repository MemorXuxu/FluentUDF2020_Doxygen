46/**********************************************************************
47 * Function prototypes:
48 *
49 * BSP_freeTreeNode
50 *
51 * Free a BSP tree node back to our own free list. The freed
52 * node is put back on the top of the list. The free list is
53 * connected through the "kid[0]" pointer.
54 *
55 * BSP_allocateTreeNode
56 *
57 * Get a BSP tree node from our own free list. If the free
58 * list is empty we try to allocate another block of nodes,
59 * and return "NULL" if another block cannot be allocated.
60 * The free list is connected through the "kid[0]" pointer.
61 *
62 * BSP_freeTree
63 *
64 * Free a BSP tree.
65 *
66 * BSP_buildTree
67 *
68 * Build a BSP tree. The function "refine()" is used to decide
69 * if a voxel needs to be refined further, and "compare()" is
70 * used to determine which kid voxel an item is in when
71 * distributing items to the kids.
72 *
73 * BSP_boxBoxIntersect
74 *
75 * Determine if there is an intersection between the axis-aligned box
76 * specified by its bounding planes "tMin[]" and "tMax[]" and the
77 * bounding box specified by its bounding planes "rMin[]" and "rMax[]".
78 *
79 * BSP_sphereBoxIntersect
80 *
81 * Determine if there is an intersection between the sphere
82 * given by its center "rc[]" and its radius "radius" and the
83 * bounding box specified by its bounding planes "rMin[]" and
84 * "rMax[]". Algorithm from Graphics Gems I.
85 *
86 * BSP_itemsInBox
87 *
88 * Find all items in voxels which are partially contained in
89 * the bounding box specified by its bounding planes "tMin[]" and "tMax[]".
90 * The function "inside()" is used to determine if the item is
91 * inside the box or not. Specifying NULL as the function
92 * pointer for "inside()" finds all items that could
93 * possibly be inside the sphere, and so requires less time
94 * since it doesn't need to call "inside()".
95 * Note that there may be duplicates in the array of items returned,
96 * since multiple voxels can contain the same item.
97 *
98 * BSP_itemsInSphere
99 *
100 * Find all items in which are contained in the sphere
101 * specified by its center "rc[]" and its radius "radius".
102 * The function "inside()" is used to determine if the item is
103 * inside the sphere or not. Specifying NULL as the function
104 * pointer for "inside()" finds all items that could
105 * possibly be inside the sphere, and so requires less time
106 * since it doesn't need to call "inside()".
107 * Note that there may be duplicates in the array of items returned,
108 * since multiple voxels can contain the same item.
109 *
110 * BSP_closestItem
111 *
112 * Find the item which is closest to the point "rc[]". The
113 * function "distance()" is used to calculate the distance
114 * between the point and a particular item. The index of
115 * the closest item is returned, or -1 if no item can be found.
116 *
117 * BSP_rayBoxIntersect
118 *
119 * Determine if there is an intersection between a ray specified
120 * by is origin "ro[]" and the unit vector "sv[]" with a
121 * bounding box specified by is bounding planes "rMin[]" and
122 * "rMax[]". The minimum and maximum intersection points
123 * along the ray are returned in "*dMin" and "*dMax".
124 *
125 * BSP_rayTreeIntersect
126 *
127 * Find the intersection point "ri[]" of a ray (specified by
128 * its origin "ro[]" and the unit vector "sv[]") with items in
129 * the tree. The function "intersect()" is used to test for
130 * intersection with leaf items and to compute the intersection
131 * point and return the item intersected.
132 **********************************************************************/
