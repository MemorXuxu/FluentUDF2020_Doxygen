snprintf
int snprintf(char *s, size_t maxlen, const char *format,...)

Write formatted output to sized buffer (safe variant of snprintf)

As opposed to snprintf() the second argument denotes the size of the destination buffer, not the number of characters to write. The result in the destination buffer is always null terminated.

Parameters
_destDestination string
_dest_lenSize of destination string buffer
_formatprintf style format followed by arguments
