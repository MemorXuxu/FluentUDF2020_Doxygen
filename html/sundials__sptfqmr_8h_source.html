13 * This is the header file for the implementation of the scaled
14 * preconditioned Transpose-Free Quasi-Minimal Residual (SPTFQMR)
15 * linear solver.
16 *
17 * The SPTFQMR algorithm solves a linear system of the form Ax = b.
18 * Preconditioning is allowed on the left (PREC_LEFT), right
19 * (PREC_RIGHT), or both (PREC_BOTH). Scaling is allowed on both
20 * sides. We denote the preconditioner and scaling matrices as
21 * follows:
22 * P1 = left preconditioner
23 * P2 = right preconditioner
24 * S1 = diagonal matrix of scale factors for P1-inverse b
25 * S2 = diagonal matrix of scale factors for P2 x
26 * The matrices A, P1, and P2 are not required explicitly; only
27 * routines that provide A, P1-inverse, and P2-inverse as operators
28 * are required.
29 *
30 * In this notation, SPTFQMR applies the underlying TFQMR method to
31 * the equivalent transformed system:
32 * Abar xbar = bbar, where
33 * Abar = S1 (P1-inverse) A (P2-inverse) (S2-inverse),
34 * bbar = S1 (P1-inverse) b, and
35 * xbar = S2 P2 x.
36 *
37 * The scaling matrices must be chosen so that vectors
38 * S1 P1-inverse b and S2 P2 x have dimensionless components. If
39 * preconditioning is done on the left only (P2 = I), by a matrix P,
40 * then S2 must be a scaling for x, while S1 is a scaling for
41 * P-inverse b, and so may also be taken as a scaling for x.
42 * Similarly, if preconditioning is done on the right only (P1 = I,
43 * P2 = P), then S1 must be a scaling for b, while S2 is a scaling
44 * for P x, and may also be taken as a scaling for b.
45 *
46 * The stopping test for the SPTFQMR iterations is on the L2-norm of
47 * the scaled preconditioned residual:
48 * || bbar - Abar xbar ||_2 < delta
49 * with an input test constant delta.
50 *
51 * The usage of this SPTFQMR solver involves supplying two routines
52 * and making three calls. The user-supplied routines are:
53 * atimes(A_data, x, y) to compute y = A x, given x,
54 * and
55 * psolve(P_data, y, x, lr) to solve P1 x = y or P2 x = y for x,
56 * given y.
57 * The three user calls are:
58 * mem = SptfqmrMalloc(lmax, vec_tmpl);
59 * to initialize memory
60 * flag = SptfqmrSolve(mem, A_data, x, b, pretype, delta, P_data,
61 * sx, sb, atimes, psolve, res_norm, nli, nps);
62 * to solve the system, and
63 * SptfqmrFree(mem);
64 * to free the memory allocated by SptfqmrMalloc().
65 * Complete details for specifying atimes() and psolve() and for the
66 * usage calls are given in the paragraphs below and in the header
67 * file sundials_iterative.h.
68 * -----------------------------------------------------------------
69 */

140/*
141 * -----------------------------------------------------------------
142 * Function : SptfqmrSolve
143 * -----------------------------------------------------------------
144 * SptfqmrSolve solves the linear system Ax = b by means of a scaled
145 * preconditioned Transpose-Free Quasi-Minimal Residual (SPTFQMR)
146 * method.
147 *
148 * mem pointer to an internal memory block allocated during a
149 * prior call to SptfqmrMalloc
150 *
151 * A_data pointer to a data structure containing information
152 * about the coefficient matrix A (passed to user-supplied
153 * function referenced by atimes (function pointer))
154 *
155 * x vector (type N_Vector) containing initial guess x_0 upon
156 * entry, but which upon return contains an approximate solution
157 * of the linear system Ax = b (solution only valid if return
158 * value is either SPTFQMR_SUCCESS or SPTFQMR_RES_REDUCED)
159 *
160 * b vector (type N_Vector) set to the right-hand side vector b
161 * of the linear system (undisturbed by function)
162 *
163 * pretype variable (type int) indicating the type of
164 * preconditioning to be used (see sundials_iterative.h)
165 *
166 * delta tolerance on the L2 norm of the scaled, preconditioned
167 * residual (if return value == SPTFQMR_SUCCESS, then
168 * ||sb*P1_inv*(b-Ax)||_L2 <= delta)
169 *
170 * P_data pointer to a data structure containing preconditioner
171 * information (passed to user-supplied function referenced
172 * by psolve (function pointer))
173 *
174 * sx vector (type N_Vector) containing positive scaling factors
175 * for x (pass sx == NULL if scaling NOT required)
176 *
177 * sb vector (type N_Vector) containing positive scaling factors
178 * for b (pass sb == NULL if scaling NOT required)
179 *
180 * atimes user-supplied routine responsible for computing the
181 * matrix-vector product Ax (see sundials_iterative.h)
182 *
183 * psolve user-supplied routine responsible for solving the
184 * preconditioned linear system Pz = r (ignored if
185 * pretype == PREC_NONE) (see sundials_iterative.h)
186 *
187 * res_norm pointer (type realtype*) to the L2 norm of the
188 * scaled, preconditioned residual (if return value
189 * is either SPTFQMR_SUCCESS or SPTFQMR_RES_REDUCED, then
190 * *res_norm = ||sb*P1_inv*(b-Ax)||_L2, where x is
191 * the computed approximate solution, sb is the diagonal
192 * scaling matrix for the right-hand side b, and P1_inv
193 * is the inverse of the left-preconditioner matrix)
194 *
195 * nli pointer (type int*) to the total number of linear
196 * iterations performed
197 *
198 * nps pointer (type int*) to the total number of calls made
199 * to the psolve routine
200 * -----------------------------------------------------------------
201 */
202
