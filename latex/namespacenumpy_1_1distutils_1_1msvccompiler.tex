{\footnotesize\ttfamily \+\_\+merge (\begin{DoxyParamCaption}\item[{}]{old,  }\item[{}]{new }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}

\begin{DoxyVerb}Concatenate two environment paths avoiding repeats.

Here `old` is the environment string before the base class initialize
function is called and `new` is the string after the call. The new string
will be a fixed string if it is not obtained from the current environment,
or the same as the old string if obtained from the same environment. The aim
here is not to append the new string if it is already contained in the old
string so as to limit the growth of the environment string.

Parameters
----------
old : string
    Previous environment string.
new : string
    New environment string.

Returns
-------
ret : string
    Updated environment string.\end{DoxyVerb}
